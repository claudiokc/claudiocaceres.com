# claudiocaceres.com

Welcome to the personal website and portfolio of Claudio Cáceres, a software engineer. This site showcases my work, projects, and technical blog, and is built with a modern React + Vite stack.

![CC Logo](public/cc-512x512.svg)

## 🚀 Tech Stack

- **Frontend:** React 18, Vite
- **Styling:** CSS Modules
- **Testing:** Vitest, React Testing Library
- **PWA Ready:** Service Worker support

## 🛠️ Available Scripts

In the project directory (`fe/`), you can run:

| Command         | Description                                 |
|-----------------|---------------------------------------------|
| `yarn dev`      | Start the development server                |
| `yarn build`    | Build the app for production                |
| `yarn preview`  | Preview the production build locally        |
| `yarn test`     | Run the test suite                          |

## 📁 Project Structure

- `src/` – Main source code (React components, styles, tests)
- `public/` – Static assets (SVG logos, favicon, manifest)
- `package.json` – Project metadata and dependencies
